## Troubleshooting

### Cannot Connect Service

**Check:**
1. âœ… Popup blocker is disabled
2. âœ… Third-party cookies are enabled
3. âœ… Service is not experiencing downtime
4. âœ… You have valid credentials for the service

**Try:**
- Use incognito/private browsing mode
- Clear browser cache and cookies
- Try a different browser

### AREA Not Executing

**Diagnose:**
1. Check if AREA is enabled (toggle switch)
2. Verify service connections are active
3. Review execution history for error messages
4. Test action parameters manually

**Common Fixes:**
- Reconnect the service
- Update action parameters
- Check API rate limits
- Verify webhook URLs

### Page Loading Issues

**Solutions:**
1. Clear browser cache: `Ctrl/Cmd + Shift + Del`
2. Disable browser extensions temporarily
3. Check internet connection
4. Try different browser
5. Check browser console for errors (`F12`)

### Data Not Syncing

**Steps:**
1. Refresh the page: `Ctrl/Cmd + R`
2. Hard refresh: `Ctrl/Cmd + Shift + R`
3. Check network tab in developer tools
4. Verify server status
5. Log out and log back in
