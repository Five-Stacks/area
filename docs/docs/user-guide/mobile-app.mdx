# 📱 Mobile Application Guide

The AREA mobile application brings the power of automation to your smartphone. Available for both iOS and Android, it provides a native mobile experience with all the features you need on the go.

## Getting Started

### Installation

#### iOS (iPhone/iPad)
1. Open the **App Store**
2. Search for "AREA Automation"
3. Tap **"Get"** to download
4. Wait for installation to complete
5. Tap **"Open"**

#### Android
1. Open the **Google Play Store**
2. Search for "AREA Automation"
3. Tap **"Install"**
4. Wait for installation to complete
5. Tap **"Open"**

:::tip
Alternatively, scan the QR code from the AREA website to download directly.
:::

### First Launch

When you first open the app:

1. **Welcome Screen**: Read the intro and tap **"Get Started"**
2. **Sign Up / Login**: Choose your preferred method:
   - 📧 Email and password
   - 🔗 OAuth (GitHub, Discord, Microsoft, etc.)
3. **Permissions**: Grant necessary permissions:
   - ✅ Notifications (recommended)
   - ✅ Network access (required)

### Connecting Services

#### Step-by-Step Process

1. **Navigate** to the Services tab
2. **Tap** on a service card (e.g., "GitHub")
3. **Tap** the **"Connect"** button
4. **Authenticate** via OAuth:
   - You'll be redirected to the service's login page
   - Log in with your credentials
   - Grant permissions requested
5. **Return** to the app automatically
6. ✅ Service is now connected!

<!-- TODO: Add screenshot - ![Connect Service Mobile](/img/user-guide/mobile-connect-service.png) -->

:::tip
Make sure to grant all requested permissions. Limited permissions may prevent some actions or reactions from working.
:::

### Managing Service Connections

#### View Service Details
Tap on a connected service to see:
- Connected account name/email
- Connection date
- Permissions granted
- AREAs using this service

#### Disconnect a Service
1. Tap on the connected service
2. Scroll down to **"Disconnect Service"**
3. Tap and confirm

:::warning
Disconnecting a service will disable all AREAs using that service!
:::

#### Reconnect with More Permissions
1. Disconnect the service
2. Connect again
3. Grant additional permissions when prompted

## AREA Tab

Create, view, and manage your automation workflows.

### Viewing Your AREAs

The AREA tab displays all your automations in a scrollable list.

**Each AREA card shows:**
- 📝 Name
- ⚡ Action (service + event)
- 🎯 Reaction (service + action)
- 🔘 Enable/disable toggle
- ⏰ Last execution time
- ✅/❌ Last execution status

<!-- TODO: Add screenshot - ![AREA List Mobile](/img/user-guide/mobile-area-list.png) -->

### Creating a New AREA

1. **Tap** the **"+"** floating action button (bottom-right)
2. **Fill in** the AREA name
3. **Select** an action:
   - Tap "Select Action"
   - Browse services
   - Choose your trigger
4. **Configure** action parameters
5. **Select** a reaction:
   - Tap "Select Reaction"
   - Browse services
   - Choose your response
6. **Configure** reaction parameters
7. **Tap** "Create" to save

### Offline Mode

The app caches data for offline viewing:
- ✅ View existing AREAs
- ✅ Check execution history
- ❌ Cannot create/edit AREAs
- ❌ Cannot trigger executions

:::tip
Changes made offline will sync when you're back online.
:::

### Widgets (iOS & Android)

Add AREA widgets to your home screen:

**iOS:**
1. Long-press on home screen
2. Tap "+" icon
3. Search "AREA"
4. Choose widget size
5. Add to home screen

**Android:**
1. Long-press on home screen
2. Tap "Widgets"
3. Find "AREA"
4. Drag to home screen

**Widget Types:**
- 📊 Quick stats

## Troubleshooting

### App won't connect

**Solutions:**
1. Check internet connection
2. Verify server status
3. Force close and reopen app
4. Clear app cache
5. Reinstall if needed

### Notifications not working

**Check:**
1. Notifications enabled in app
2. System permissions granted
3. Do Not Disturb mode off
4. Background refresh enabled

### Service connection fails

**Try:**
1. Disconnect and reconnect service
2. Clear browser cache (for OAuth)
3. Check service status page
4. Try different network

### App crashes or freezes

**Steps:**
1. Force close the app
2. Clear app cache
3. Update to latest version
4. Restart your device
5. Reinstall if problem persists

### Sync issues

**Resolve:**
1. Check internet connection
2. Pull to refresh
3. Log out and log back in
4. Clear cache and data

## Best Practices

### 🔋 Battery Optimization

The app is optimized for battery life, but you can:
- Reduce sync frequency
- Disable background refresh for non-critical AREAs
- Use WiFi instead of mobile data

### 📊 Data Usage

Minimize data consumption:
- Enable WiFi-only sync
- Reduce notification frequency
- Download execution logs on WiFi

### 🔐 Security

Protect your automations:
- Enable biometric lock
- Use strong password
- Review connected services regularly
- Log out on shared devices

### ⚡ Performance

Keep the app running smoothly:
- Update regularly
- Clear cache monthly
- Disable unused AREAs
- Remove old execution history
