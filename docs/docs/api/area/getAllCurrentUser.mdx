## GET `/api/area/`

Retrieve all areas for the current user.

### Response
- **200 OK**
  ```json
  { "success": true, "data": [ /* array of areas */ ] }
  ```
- **500 Internal Server Error**
  ```json
  { "success": false, "error": "Internal Server Error" }
  ```

### Area object

```json
{
  "id": 1,
  "user_id": 1,
  "action_id": 1,
  "reaction_id": 2,
  "config": {},
  "is_active": true,
  "created_at": "2024-06-10T12:00:00.000Z"
}
```

### Example
```bash
curl -X GET http://localhost:8080/api/area/ \
  --cookie "token=your_jwt_token"
```
