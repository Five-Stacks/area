## GET `/api/area/:id`

Retrieve a specific area by its ID (for the current user).

### Response
- **200 OK**
  ```json
  { "success": true, "data": { /* area object */ } }
  ```
- **404 Not Found**
  ```json
  { "success": false, "error": "Area not found" }
  ```
- **500 Internal Server Error**
  ```json
  { "success": false, "error": "Internal Server Error" }
  ```

### Area object

```json
{
  "id": 1,
  "user_id": 1,
  "action_id": 1,
  "reaction_id": 2,
  "config": {},
  "is_active": true,
  "created_at": "2024-06-10T12:00:00.000Z"
}
```

### Example
```bash
curl -X GET http://localhost:8080/api/area/1 \
  --cookie "token=your_jwt_token"
```
